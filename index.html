Bistro | Programa de Reserva de Mesas | C

Repositório Bistro em C

Durante minha formação em Análise e Desenvolvimento de Sistemas, desenvolvi um projeto em linguagem C que simula um sistema de pedidos para um restaurante. O objetivo da atividade foi aplicar conceitos de programação estruturada, lógica de programação e manipulação de dados em um contexto prático.

Este projeto reforçou minhas habilidades em desenvolvimento de software, resolução de problemas e boas práticas de programação, além de demonstrar minha capacidade de transformar conceitos teóricos em soluções funcionais.
